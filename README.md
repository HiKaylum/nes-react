# react-nes

Install

```
npm i @kylum/nes-react ness.css @fontsource/press-start-2p
```

Import `App.js`

```
import "@fontsource/press-start-2p";
import "nes.css/css/nes.min.css";
```

Example importing component

```
import { Button } from '@kylum/nes-react';
```
